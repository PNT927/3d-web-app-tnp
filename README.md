# Philip 3D Web Application
## Overview
This is an interactive 3D web application built for the 3D Web Application assignment. The idea behind it was to create something engaging in the browser using Three.js, with three drink models I made myself in Blender a Coke, a Fanta and a Sprite. You can switch between the models and play around with them using a few different controls, including animation playback, lighting adjustments, wireframe mode and a camera reset.

## Features
- Interactive 3D model viewer
- GLB model loading using Three.js
- OrbitControls for camera movement
- Model switching system
- Animation playback
- Wireframe toggle
- Lighting toggle
- Camera reset feature
- Responsive Bootstrap layout
- About page
- Submission page

## Running the Project

**Option 1 — VS Code Live Server**
1. Open the project folder in VS Code
2. Right click `index.html`
3. Select **Open with Live Server**
4. The project should open in the browser automatically

---

## Controls

| Button | Function |
|--------|----------|
| Coke / Fanta / Sprite | Switch between models |
| Start Spin | Start model rotation and animation |
| Stop Spin | Stop model rotation and animation |
| Wireframe | Toggle wireframe mode |
| Light On/Off | Toggle scene lighting |
| Reset Camera | Return camera to default position |
