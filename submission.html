Deeper Understanding

This project goes beyond the basic lab template to show a deeper understanding of the concepts involved. I redesigned the interface, organised the model folders, and built out reusable JavaScript functions to keep things clean. From there, I added GLB model loading, OrbitControls, animation playback, wireframe mode, lighting controls and model switching. I also threw in a camera reset feature so users can quickly snap back to the default view after moving around the 3D scene.
